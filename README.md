# Starline Landing

Static one-page landing prototype for Starline, an e-commerce growth operator.

## Local Preview

Run:

```bash
python3 -m http.server 4173 --bind 127.0.0.1
```

Open:

```text
http://127.0.0.1:4173/
```

## Contents

- `index.html` - landing page markup.
- `landing/styles.css` - responsive visual styling.
- `landing/assets/slides/` - selected visual assets rendered from the Starline source presentation.
