# Starline Landing

Static one-page landing prototype and Codex skill for Starline, an e-commerce growth operator.

## Landing Preview

Run:

```bash
python3 -m http.server 4173 --bind 127.0.0.1
```

Open:

```text
http://127.0.0.1:4173/
```

## Contents

- `index.html` - landing page markup.
- `landing/styles.css` - responsive visual styling.
- `landing/assets/slides/` - selected visual assets rendered from the Starline source presentation.
- `SKILL.md` - Codex skill entrypoint.
- `references/starline-brand.md` - Starline positioning, copy, service, and caveat reference.
- `assets/starter/` - copyable starter version of the landing prototype for skill users.
